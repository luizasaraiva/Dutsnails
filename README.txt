DutsNails — Projeto estruturado no estilo DynamicDoc

COMO ABRIR
1. Abra o arquivo index.html no navegador.
2. O site público funciona sem login.
3. O botão Entrar abre acesso para cliente ou administradora.

ESTRUTURA
- index.html: página principal do site.
- css/style.css: aparência visual do projeto.
- js/main.js: funções gerais, serviços e galeria.
- js/agenda.js: envio da reserva para WhatsApp.
- js/auth.js: login de cliente e administradora.
- js/studio.js: funções do painel administrativo.
- data/config.js: WhatsApp, nome da marca e cidade.
- data/servicos.js: lista de serviços, valores, duração e observações.
- data/galeria.js: imagens/categorias da galeria.
- cliente/: telas do Clube Duts.
- studio/: telas do painel administrativo.
- assets/portfolio/: pasta para guardar fotos reais da galeria.

ONDE EDITAR MAIS FÁCIL
- Alterar preços/serviços: data/servicos.js
- Alterar WhatsApp/cidade: data/config.js
- Alterar fotos da galeria: data/galeria.js
- Alterar cores/layout: css/style.css
